# UX Research

Journey maps y storyboards interactivos del flujo de arrendamiento.

| Artefacto | Tipo | Descripción |
|-----------|------|-------------|
| [Journey · José lista propiedad](journey-jose-listado-propiedad/index.md) | Customer Journey Map | José (propietario) crea un listado y publica su propiedad |
